
# LOSI Municipal Portal Scanner

Experimental research tool for scanning municipal portals and detecting selected LOSI indicators.

## Indicators implemented

| Code | Name | Description |
|-----|-------------|-----|
| 204 | Mobile device accessibility (viewport meta) | Evidence of a “contact us” feature (e.g. phone numbers, email addresses, physical addresses, and sometimes online forms) |
| 205 | Navigability | A sitemap or index with a list of all the available pages within the website accessible to users. |
| 206 | Internal search mechanism | A search bar on the main web page of the municipal government portal. |
| 214 | Contact details | Evidence of a “contact us” feature (e.g. phone numbers, email addresses, physical addresses, and sometimes online forms) |
| 234 | Privacy policy | Existence of a privacy policy or statement available on the MGP |
| 273 | Social networking features | The provision of social networking feature(s) (e.g. Facebook, Twitter, YouTube, Flickr, etc.). |

This application is a work in progress developed within the Postgraduate Programme "e-Government" of the University of the Aegean. It aims to audit municipal websites using the Local Online Service Index (LOSI) indicators.
See the tool in action: https://losi-scanner.streamlit.app/
