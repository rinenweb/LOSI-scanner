
# LOSI Municipal Portal Scanner

Experimental research tool for scanning municipal portals and detecting selected LOSI indicators.

## Indicators implemented

| Code | Description |
|-----|-------------|
| 206 | Internal search mechanism |
| 214 | Contact details |
| 234 | Privacy policy |
| 273 | Social networking features |
| 204 | Mobile device accessibility (viewport meta) |

## Run locally

Install dependencies:

pip install -r requirements.txt

Run the interface:

streamlit run app.py
